# Power-BI-Pizza-Sales-Analysis
🍕 Pizza Sales Analysis
This project is a sales analysis of a pizza store using Power BI. I created different visuals and dashboards to understand the sales performance and customer trends.

📄 What’s Included
  In this analysis, I looked at:

    - Total revenue from pizza sales

   - Most and least sold pizzas

   - Sales by day, week, and month

   - Peak order times during the day

  - Performance of different pizza sizes and categories

🛠 Tools Used
 -  Power BI for creating visuals and dashboards.
 -  Data was cleaned and analyzed directly in Power BI using Power Query.

🎯 Purpose
    I made this project to practice Power BI and to learn how to work with sales data. It helped me understand how to find insights and show them clearly using visuals.
